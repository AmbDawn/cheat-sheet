CSS Font Cheat Sheet

Spark Your Memory

Overview

It is extremely important to remember how to style fonts for any website project. These changes will make or break the feels and professionalism of any site.

The main points to spark memory for are:

  • Families
  • Styles
  • Alignment

Families

When a font or typeface has been selected, there are 3 options to display the fonts correctly on any websites:

  • Use web-safe fonts
  • Link to an online font-source api
  • Use local fonts
Import Options Example Code
Web-Safe Fonts font-family: Arial, sans-serif;
Link to Online Font <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
Local Fonts @font-face {
font-family: "Dotness";
src: url("/resources/fonts/dotness.ttf");
}

Styles

There are 7 main text styles that can be applied to any font-family:

  • Font-Style
  • Font-Weight
  • Font-Size
  • Text-Transform
  • Text-Decoration
  • Color
  • Background-Color
Styles Example Code
Font-Style font-style: italic, normal, oblique;
Font-Weight font-weight: normal, bold, bolder, lighter, 100-900;
Font-Size font-size: 16px, 1em, 100%;
Text-Transform text-transform: none, uppercase, lowercase, capitalize;
Text-Decoration text-decoration: none, underline, overline, line-through, blink;
Color color: #000000, rgb(0,0,0), rgba(0,0,0,1), hsl(0,0,0), hsla(0,0,0,1);
Background-Color background-color: #ffffff, rgb(255,255,255), rgba(255,255,255,1), hsl(0,0,100), hsla(0,0,100,1);

Alignment

There are 4 main text alignment options that can be applied to any font-family:

  • Text - Align
  • Vertical - Align
  • Line - Height
  • Letter - Spacing
  • Word - Spacing
Alignment Code
Text-Align text-align: left, right, center, justify; (to parent element)
Vertical-Align vertical-align: baseline, top, middle, bottom; (to parent element)
Line-Height line-height: 1.5, 150%, 24px;
Letter-Spacing letter-spacing: 1px, 0.1em;
Word-Spacing word-spacing: 1px, 0.1em;
